## ****프로젝트 핵심 목적****

- 클론코딩에서 요구하는 핵심기능을 연습
-  Generic이 아닌 def view로 작성 연습

## ****🎈 기능 명세서****

- HTML/CSS/Javascript를 활용한 프론트엔드 페이지 구성
- 사용자 환경(회원가입, 로그인, 로그아웃 등)
- 게시글 작성, 보여주기, 수정, 삭제 기능 구현(사진업로드)
- 댓글 작성, 보여주기, 수정, 삭제 기능 구현
- 좋아요, 팔로우 기능 구현
